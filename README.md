# simple-feedback-message-box

Character Restriction Textarea

A simple web project built using HTML, CSS, and JavaScript that limits user input in a textarea to a maximum of 200 characters.

Features
User-friendly textarea input
Maximum character limit of 200 characters
Real-time character counting
Alert popup when the character limit is reached
Prevents users from exceeding the allowed character count
Clean and responsive design
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
How It Works
The user types text into the textarea.
JavaScript continuously monitors the number of characters entered.
When the user reaches the 200-character limit:
An alert popup appears with the message:
"You have reached the maximum character limit!"
Additional characters are prevented from being entered.
The user can still edit or remove existing text using the Backspace key.

Project Purpose

This project was created to improve JavaScript fundamentals by implementing a real-world form validation feature commonly used in contact forms, feedback forms, and social media text inputs.
